# Amazon Prime Movies & TV Shows EDA

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Amazon Prime Movies & TV Shows dataset using Python.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset

- titles.csv
- credits.csv

## Key Insights

- TV Shows have higher IMDb ratings than Movies.
- Drama is the dominant genre.
- The US contributes the largest number of titles.
